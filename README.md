# Bithues Basic Website

Simple static site for bithues.com.

## Files
- `index.html` — Single-page website

## To Deploy
Upload the entire `bithues-basic/` folder to Cloudflare Pages.
- Build command: (none)
- Output directory: .

## Design
- Dark theme with purple accent
- Responsive (mobile-friendly)
- No external dependencies
- Single HTML file

## Customization
Edit `index.html` to change:
- Title and tagline
- Card content
- Colors (CSS variables at top)
