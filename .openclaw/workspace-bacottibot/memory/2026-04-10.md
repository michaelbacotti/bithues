## Key Lessons — Accuracy, Subagents, and My Behavior

**Incident (today):**
- Mike reported blank black screen. I assumed browser cache issue without checking.
- Subagent pushed "fixed" code that had a JavaScript error: `.reduce()` called on a plain object — game crashed silently
- Mike saw the actual browser console error and showed me via screenshot. I had been claiming it worked fine in my headless browser.
- This happened MULTIPLE TIMES today with AutoDuel: I asserted things were correct without reading the actual code.

**The pattern:**
1. Mike reports a problem
2. I immediately speculate about causes (cache, browser, etc.)
3. I don't actually read the relevant code to verify
4. Subagents build on my unverified assumptions → deliver broken code
5. Mike ends up catching what I should have caught

**Subagent performance issues today:**
- Subagent timed out while mid-visual-rewrite — delivered nothing useful
- Subagent spent tokens reading PDFs it barely processed
- Multiple subagents needed to redo work because first one timed out
- One subagent produced a commit that crashed the game on load

**What needs to change:**
1. **Before claiming "done" or "fixed":** Actually read the relevant lines of code. Not "I think it's fine" — read the actual source.
2. **Take Mike's browser reports seriously.** He has eyes on the actual game. My headless browser is not the ground truth.
3. **Give subagents smaller, one-shot scopes.** One bug per subagent. If it times out, it delivered nothing.
4. **Verify subagent output before claiming success.** Check the pushed file size, check the code, take a screenshot.
5. **Slow down.** Mike explicitly prefers quality and accuracy over speed.

**Skills to consider creating/improving:**
- `skill-creator` → create a "subagent-instruction-builder" that forces smaller scopes
- `skill-creator` → create a "code-verifier" that prompts me to read actual lines before claiming fixed
- Existing skills that need accuracy reminders: `bot-manager`, `subagent-auditor`

**Note to self:** This is not the first time accuracy has been flagged as an issue. See also: MEMORY.md "Critical Lesson — Accuracy in HOUSE Inc Cases" (DeAutra Carter incident from 2026-04-08). These incidents are becoming a pattern.
