### The fundamental failure
**Never diagnose a site problem by looking at files — always check the live URL first.**

```bash
# Always do this FIRST when debugging any site issue:
curl -s "https://www.sitename.com/problem"
curl -s -o /dev/null -w "%{http_code}" "https://www.sitename.com/problem"
```
