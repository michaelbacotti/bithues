# SKILL.md — kb-accounting-review

## What it does
Reviews and organizes accounting records for Bacotti family business entities, reconciles accounts, and identifies gaps in financial statements.

## When to use
- "Accounting review", "reconcile", "financial statements", "bookkeeping"
- Monthly/annual reconciliation needs
- Identifying missing P/L entries or financial documents

## When NOT to use
- For tax filing preparation (use kb-tax-preparation)
- For trading decisions (use options-pro)
- For entity overview questions (use kb-entity-overview)

## Inputs
- Entity folders and reference documents
- Dependability P/L tracker CSV
- Tax documents in entity reference folders

## Outputs
- Summary of financial status per entity
- Gap analysis (missing documents, unreconciled items)
- Action items ranked by urgency

## Cost / Risk notes
No external calls. Read-only review of workspace files.
