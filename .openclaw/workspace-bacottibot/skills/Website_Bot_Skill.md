# Website Bot Skill

## Purpose
- Automate the management of website-related tasks as outlined in the Website SOP.

## Features
1. **Page Creation Automation**:
   - Assist in generating new pages based on templates.

2. **Content Updates**:
   - Automate the process of checking and updating content.

3. **SEO Optimization**:
   - Provide actionable insights for improving SEO.

4. **Bug Reporting**:
   - Help identify and report bugs efficiently.

5. **Publishing Notifications**:
   - Notify when updates are pushed or published.
