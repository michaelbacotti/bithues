# SKILL.md — options-pro

## What it does
Provides stock market analysis, options trading strategy, and portfolio analysis for Dependability Holding LLC.

## When to use
- Stock or index analysis (SPX, XSP, QQQ, etc.)
- Options strategies (spreads, iron condors, verticals)
- Greeks analysis questions
- Trading decisions or market research
- User says "Options", "Trading strategy", "SPX", "Iron condor"

## When NOT to use
- For non-trading entity questions (use relevant entity skill)
- For basic stock lookup without strategy component
- For tax filing questions (use kb-tax-preparation)

## Inputs
- Market data via web search
- User's specific trade requests or analysis questions
- Current price levels and trends

## Outputs
- Market conditions summary
- Technical levels (support/resistance)
- Recommended trade with strikes, expiry, max profit/loss, rationale

## Cost / Risk notes
Web search for market data. No trading execution. Always confirm with user before executing recommendations.
