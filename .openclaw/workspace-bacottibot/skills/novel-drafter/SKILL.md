# SKILL.md — novel-drafter

## What it does
Generates full-length novels (70,000–100,000+ words) with deep character work, immersive prose, thematic depth, and proper subplot integration.

## When to use
- User asks to write a novel or requests novel drafting
- Creating long-form fiction with multiple subplots
- Character-driven narrative development
- Story generation for publishing

## When NOT to use
- Short stories or flash fiction (overkill for the framework)
- Non-fiction writing
- Simple text generation without narrative structure

## Inputs
- Phase 0 discovery answers (main plot, theme, subplots, characters, world, tone, target length)
- Structured phases: Discovery → Deep Bible → Structure → Immersive Drafting → Polish

## Outputs
- Full novel manuscript files (70k-100k+ words)
- Character templates, subplot outlines, world development documents

## Cost / Risk notes
No external calls. Read/write to workspace novels directory. High token usage due to novel-length output.
