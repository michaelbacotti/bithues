# SKILL.md — kb-maintenance

## What it does
Keeps the Bacotti Knowledge Base (KB) up to date — after significant sessions, updates entity files, meeting minutes, financial trackers, and reference documents.

## When to use
- "Update KB", "update knowledge base", "save meeting", "record decision"
- End of significant work session
- After any entity-related work (trading, properties, taxes)

## When NOT to use
- For one-off trivial tasks
- For queries that don't create lasting context
- When workspace is already current

## Inputs
- Session outcomes and decisions
- Entity data, meeting notes, financial updates
- KB folder structure in `References/Bacotti Enterprises/`

## Outputs
- Updated entity files, meeting minutes, financial trackers
- Notes added to daily memory file
- Follow-up items flagged

## Cost / Risk notes
No external calls. Read/write to workspace entity folders and memory files.
