# SKILL.md — website-bot

## What it does
Creates, updates, and manages websites using HTML/CSS/JavaScript, with GitHub Pages deployment support.

## When to use
- Creating new websites from scratch
- Updating existing website content
- Working with HTML, CSS, JavaScript files
- GitHub Pages deployment
- Website troubleshooting and bug fixes

## When NOT to use
- For content proofreading/bug checking — use LOCAL files first (not live site fetches)
- Backend-only tasks without frontend components
- Tasks requiring server-side processing

## Inputs
- Project name and specifications
- Local files in the workspace (e.g., `~/.openclaw/workspace-bacottibot/` for bithues) — source of truth, not live URLs

## Outputs
- HTML, CSS, JavaScript files
- GitHub repo creation and GitHub Pages deployment

## Cost / Risk notes
Writes to workspace and GitHub. Live site fetches should be avoided — local files are source of truth.
