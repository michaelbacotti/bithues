# SKILL.md — kb-entity-overview

## What it does
Provides comprehensive overview of all Bacotti family business entities — ownership structure, financial status, key contacts, and filing requirements.

## When to use
- "Entity", "entities", "business structure", "ownership", "overview"
- Questions spanning multiple entities
- Understanding relationships between Bacotti Inc., Dependability Holding, Succession Holding, HOUSE Inc.

## When NOT to use
- For deep-dive expertise on specific entity (use kb-dependability-holding, kb-succession-holding, etc.)
- For tax filing details (use kb-tax-preparation)
- For trading questions (use options-pro)

## Inputs
- Entity folders: dependability-holding-llc/, succession-holding/, bacotti-inc/, house-inc/
- Reference documents in `References/Bacotti Enterprises/`

## Outputs
- Complete entity status summary with financials, filing status, key contacts, action items

## Cost / Risk notes
No external calls. Read-only reference to workspace files.
