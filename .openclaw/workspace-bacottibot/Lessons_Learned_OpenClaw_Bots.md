# Lessons Learned for OpenClaw Bots

## Model Performance
- **Minimax**: Best performance during the trial period. Great accuracy, intelligent, and autonomous.
- **Ollama**: Relative poor performance.
- **Gemini**: Average reliability, but had some inconsistencies.
- **GPT**: Good performance overall, but made organizational mistakes that Minimax did not. More attention needed on accuracy rather than speed.

## Search Performance
- **Gemini**: Performance was acceptable, but used minimally.
- **Brave**: Early impressions are positive, but has just started.
