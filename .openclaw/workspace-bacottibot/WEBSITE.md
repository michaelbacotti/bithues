## Bithues Website Local Repos

- The actual website folders live under ~/Sites, not under the iCloud workspace path.
- Relevant folders:
 - ~/Sites/bithues-lab (primary Bithues Reading Lab site)
 - ~/Sites/bithues-temp (temporary or staging variant, if used)
- When the user asks to work on, update, or check the Bithues website (including ):
 - Treat ~/Sites/bithues-lab as the main repo by default.
 - Change directory to ~/Sites/bithues-lab before running git or npm commands, unless the user explicitly says to use bithues-temp.
